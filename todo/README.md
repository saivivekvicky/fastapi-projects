# 📝 FastAPI To-Do App

A simple CRUD-based To-Do API built with **FastAPI + SQLAlchemy + SQLite**.

## 🚀 Features
- Create new tasks
- List all tasks
- Update tasks (title / mark complete)
- Delete tasks

## 📦 Installation
```bash
git clone https://github.com/yourusername/fastapi-todo.git
cd fastapi-todo
pip install -r requirements.txt
